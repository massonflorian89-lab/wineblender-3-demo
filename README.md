# WineBlender 3 — Démonstration

Environnement de **démonstration** de WineBlender 3, logiciel de traçabilité de cave.

🌐 **URL de la démo** : https://florianmasson.github.io/wineblender-3-demo/ *(disponible après déploiement GitHub Pages)*

> **L'accès est sur invitation uniquement.** Si vous êtes œnologue intéressé pour tester, contactez Florian.

## Ce que la démo contient

- Une **cave fictive** (`Domaine du Test`) avec 3 millésimes d'historique cohérent — cuves, lots, opérations, analyses, produits œnologiques.
- L'**intégralité des fonctionnalités** de WineBlender 3 (cuverie vivante, fiches lots, opérations verrouillées, dossier de lot PDF, assemblages, etc.).
- Les testeurs **partagent la même cave** et peuvent voir les modifications les uns des autres.

## Comment ça marche

1. Vous recevez un email d'invitation.
2. Vous vous connectez via **lien magique** (pas de mot de passe à retenir).
3. Vous explorez la cave fictive — créez des opérations, faites des assemblages, générez des dossiers de lot…
4. Vous laissez votre retour via le formulaire de feedback.

## Stack technique

Vanilla JS · Supabase (Postgres EU) · PWA installable · sans framework.
